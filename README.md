## 구현 범위

### 구현 세부
- 코디 추천 서비스 구현
    - 코디 추천(고객용) 기능 구현
    - 코디 관리(관리자) 기능 구현
- 데이터베이스 설계 및 구현, 관리 기능 추가
    - api에 특화된 조회 기능 구현
    - 사전 데이터
        - 미리 적재해둠
        - 원활한 테스트를 위해 이 데이터는 수정 및 삭제 불가

## 테스트 및 빌드
./gradlew build

## 실행
./gradlew bootRun

## 기타 정보
- 프론트 페이지
    - 고객용 api(3개), 관리자 api(3개) 테스트 가능
    - 주소: http://localhost:8080/
    - 브랜드 이름은 영문 소문자만 가능
    - 모든 가격은 100 ~ 999999 사이만 가능
- http 요청 샘플 -> utils 패키지 참고
- test -> src/test 패키지 참고
- 자바 17버전, 스프링 부트 3.3.0 버전
